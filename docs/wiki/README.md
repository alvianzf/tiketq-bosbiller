# 📚 TiketQ Bosbiller Wiki

Welcome to the internal documentation for the **TiketQ Bosbiller** project. This project serves as the backend service for flight bookings, ferry services, and car rentals within the TiketQ ecosystem.

## 🚀 Project Goals

The Bosbiller service is designed to:
1.  **Aggregrate Services**: Provide a unified interface for flight, ferry, and car rental bookings.
2.  **Ensure Reliability**: Handle real-time availability and secure payment processing.
3.  **Scalable Auth**: Centralized user and admin authentication.
4.  **Developer Friendly**: Documented patterns for adding new services and maintaining existing ones.
